# Security Policy

## Reporting

Report vulnerabilities privately to `jesuorobonosakhare873@gmail.com`.

Include:

- affected commit/hash
- reproduction steps
- impact assessment

## Scope

In scope:

- `src/` smart contracts
- deployment and demo scripts affecting protocol state
- accounting and ownership invariants

Out of scope:

- local-only demo UI simulation errors without on-chain impact

## Disclosure

Please allow coordinated disclosure and remediation before public release.
